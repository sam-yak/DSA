# 🏋️ Coaching Protocol — How Each Day Works

## When You Start a New Day

Just come in and say: **"Day X — let's go"**

I'll immediately:
1. Review your previous day (if applicable)
2. Lay out today's plan: concepts, patterns, problems
3. Teach you the theory with examples
4. Give you problems one at a time

## During Problem Solving

For each problem, here's how we'll work:

### Step 1: I give you the problem
- Problem statement, constraints, examples
- I'll ask you guiding questions before you code

### Step 2: You think out loud
- Tell me your approach BEFORE writing code
- State the data structure, time complexity, space complexity
- I'll redirect you if you're heading toward a suboptimal path

### Step 3: You write the solution
- Paste your code here
- I'll review it like a Google interviewer would

### Step 4: I coach you
- Fix bugs and logic errors
- Point out edge cases you missed
- Show you cleaner / more Pythonic ways
- If you're stuck (20 min), I'll teach you the pattern
- Then you re-implement from scratch

### Step 5: We mark progress
- ✅ Solved clean (first try, optimal)
- ⚠️ Solved with hints (needed nudges)
- ❌ Couldn't solve (learned from reference)

## At End of Day

- We update the progress tracker
- We update your GitHub repo files
- I quiz you on the day's patterns from memory
- I give you what to review before sleep

## Communication Rules

1. **Don't just paste code silently.** Tell me your thinking.
2. **If you're stuck, tell me WHERE you're stuck** — not just "I don't know"
3. **After seeing a solution, close it and re-do it.** I'll ask you to.
4. **I'll be tough but fair.** Google interviewers won't hold your hand.

## Quick Commands

| Say This | I'll Do This |
|----------|--------------|
| "Day X — let's go" | Start that day's session |
| "Teach me [pattern]" | Deep-dive explanation with examples |
| "Quiz me" | Rapid-fire pattern recognition questions |
| "Review my code" | Google-interviewer-style code review |
| "I'm stuck" | Guided hints (not the answer) |
| "Show me the solution" | Full walkthrough + re-implement prompt |
| "End of day" | Progress update + quiz + GitHub prep |
| "Pattern drill" | Give you a problem, you name the pattern in 30 sec |
